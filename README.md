# event-master
An android application to organize events in college.

## Use of application
An android application to organize events in college.

## Technologies used
1] Kotlin Language
2] Room Library
3] Retrofit Library
4] Hilt
5] Mongo DB Database
6] Express + Mongoose
7] Node JS
8] Google Maps API
9] Google Places API
10] Android Studio
11] VS Code
